fun build(dialogTitle: String, minDate: LocalDate? = null, maxDate: LocalDate? = null, selectedDate: LocalDate? = null, onDateSelected: (LocalDate) -> Unit, absoluteMinDate: LocalDate = ABSOLUTE_MIN_DATE): MaterialDatePicker<Long>

Builds a MaterialDatePicker dialog with the specified configuration.

Return

A configured instance of MaterialDatePicker.

Parameters

dialogTitle

The title to be displayed on the date picker dialog.

minDate

The minimum selectable date in the date picker. If null, defaults to absoluteMinDate.

maxDate

The maximum selectable date in the date picker. If null, there is no upper limit.

selectedDate

The initial date to be selected in the date picker. If null, no date is preselected.

onDateSelected

A callback that is invoked when the user selects a date. The selected LocalDate is passed as an argument to this callback.

absoluteMinDate

The absolute minimum date that can be selected, defaults to ABSOLUTE_MIN_DATE.
