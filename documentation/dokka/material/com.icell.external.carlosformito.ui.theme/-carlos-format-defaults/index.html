class CarlosFormatDefaults(val dateFormatter: DateTimeFormatter = DateTimeFormatter.ISO_LOCAL_DATE, val timeFormatter: DateTimeFormatter = DateTimeFormatter.ofPattern("HH:mm"), timeFormat: Int? = null)

A class that provides default formatting options for date and time values used by the Carlos form fields.

Parameters

timeFormat

An optional parameter to specify the time format, either 12-hour or 24-hour clock. If null, the system's time format will be used.
