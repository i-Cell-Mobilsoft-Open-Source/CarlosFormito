CarlosDatePicker

@Composable
fun CarlosDatePicker(dialogTitle: String, dialogHeadline: String? = null, selectedDate: LocalDate?, minDate: LocalDate? = null, maxDate: LocalDate? = null, formatter: DateTimeFormatter, onSelectDate: (LocalDate?) -> Unit, hideDialog: () -> Unit)

A composable function that displays a date picker dialog.

This function shows a DatePickerDialog with the provided title, optional headline, and selectable date range. The selected date is formatted using the provided formatter and passed to the onSelectDate callback upon confirmation. The dialog can be dismissed using the hideDialog callback.

Parameters

dialogTitle

The title of the date picker dialog.

dialogHeadline

An optional headline to display above the date picker.

selectedDate

The initially selected date. If null, no date is initially selected.

minDate

The minimum selectable date. If null, defaults to ABSOLUTE_MIN_DATE.

maxDate

The maximum selectable date. If null, there is no maximum date restriction.

formatter

The DateTimeFormatter used to format the selected date for display.

onSelectDate

Lambda function to be invoked when a date is selected. Receives the selected date as a parameter.

hideDialog

Lambda function to be invoked to dismiss the dialog.
